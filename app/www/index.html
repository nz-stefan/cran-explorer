Package Explorer

Select a package from the table to display its dependecies or references as an ego network graph. Flip the "Layout" switch to activate the online layout algorithm for a better visualisation of the network. Hovering over a network node reveals the package title and clicking of it shows the ego graph of the clicked package. Move in the graph and zoom by dragging and using the mouse wheel.
